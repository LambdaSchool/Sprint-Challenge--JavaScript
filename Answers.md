1. Describe the biggest difference between `.forEach` & `.map`.
The forEach() method executes a provided function once for each array element. While, the .map() method actually creates a new result with the results of calling the provided function for every element. In other words, .map() and .forEach() do the same thing with .map() returning a new array.

2. What is the difference between a function and a method?
A function is code that can be called by other code or by itself. It is also an object in JavaScript. A method is a function which is a property of an object. In other words, a method is an object reference to a function.

3. What is closure?
A closure consists of the lexical environment in which a function is declared.

4. Describe the four rules of the 'this' keyword.
Global Binding: is when "this" will be the window/console object.
Implicit Binding: is when "this" refers to the object on the left of the dot.
New Binding: is when "this" refers to a specific instance of the object that is created
Explicit Binding: is when "this" is implied by the use of a "call" or "apply" method.
